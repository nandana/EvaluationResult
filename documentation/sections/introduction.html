1. Introduction back to ToC

This document presents the Evaluation Result ontology, an ontology created for describing knowledge about the results obtained in an evaluation process. The ontology focuses on modelling quality values obtained in an evaluation and their related information. This document specifies the classes, properties and their relationships of the Evaluation Result ontology.
